### Multipurpose Internet Mail Extensions (MIME) 
Allows browsers to identify the file's type. In HTML, when you upload a file sometimes it will be necessary to specify this parameter:
```html
<link rel="icon" href="imgs/image.png" type="image/png"/>
```
Here we can see the specification in `type="image/png`.
